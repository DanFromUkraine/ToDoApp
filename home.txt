### Створення базового додатку "Список справ" (To-Do List)

**Мета:** Закріпити основи створення компонентів, стану (state) та обробки подій у React.

**Опис завдання:**

1. **Створіть компонент `ToDoApp`**:
   - Це буде основний компонент, який міститиме логіку додатку.

2. **Додайте компонент `ToDoList`**:
   - Цей компонент буде відображати список завдань.
   - Використовуйте `props` для передачі списку завдань до цього компонента.

3. **Додайте компонент `ToDoItem`**:
   - Цей компонент буде відображати одне завдання.
   - Використовуйте `props` для передачі тексту завдання та інформації про те, чи виконане завдання.

4. **Реалізуйте можливість додавання нових завдань**:
   - Додайте поле вводу та кнопку для додавання нових завдань до списку.
   - Використовуйте стан (`state`) для управління поточним списком завдань.

5. **Реалізуйте можливість видалення завдань**:
   - Додайте кнопку для видалення завдання поруч з кожним завданням у компоненті `ToDoItem`.
   - При натисканні на кнопку завдання має бути видалене зі списку.

6. **Реалізуйте можливість відзначати завдання як виконані**:
   - Додайте можливість натискання на завдання, щоб відзначити його як виконане або невиконане.
   - Використовуйте стан для управління станом кожного завдання (виконане/невиконане).

**Критерії оцінки:**
- Коректне використання компонентів та їх взаємодія через `props`.
- Управління станом додатку через `state`.
- Обробка подій (додавання, видалення та відзначення завдань).
- Код має бути чистим та структурованим.
